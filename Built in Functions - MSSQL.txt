                                                      Build in Functions 


Aggregate functions - Perform a calculation on a set of values and return a single value.
Examples: Average, Min, Max, Sum, Count

Analytic functions - Compute an aggregate value based on a group of rows
They return multiple rows for each group.

Ranking functions - Return a ranking value for each row in a partition.
Examples: RANK, ROW_NUMBER, DENSE_RANK, NTILE 

	-ROW_NUMBER - always generate unique values without any gaps, even if there are ties.
	-RANK - can have gaps in its sequence and when values are the same, they get the same rank.
	-DENSE_RANK - returns the same rank for ties, but it doesn't have any gaps in the sequence
	-NTILE - Distributes the rows in an ordered partition into a specified number of groups
	
Rowset functions - Return an object can be used like table references in a statement.
Example: OPENDATASOURCE, OPENJASON, OPENXML, OPENROWSET

Scalar functions - Operate on a single value and then return a single value.


1. String Functions
	- Concatenation - Combines strings. Concat replace NULL values with empty string
	We can use + symbol to concate strings and CONCAT. The best way is to use CONCAT_WS
	- Substring - Extracts a part of a string
	- Replace - replaces a specific string with another - REPLACE(String, Pattern, Replacement), REPLACE('SoftUni', 'Soft', 'Hard')
	- LTRIM & RTRIM - Remove spaces from either left or right side.
	- LEN - counts the number of characters
	- LEFT & RIGHT - get characters from the beginning or the end of string
	- LOWER & UPPER - change the letter casting 
	- REVERSE - Reverse the string
	- REPLICATE - Repeat a string
	- FORMAT - format a value with a valid .NET format string
	- CHARINDEX - locates a specific pattern (substring) in string
	- STUFF - inserts a substring at a specific position in other string
	                     -I am not sure that I will use many of them in the future.

2. Math Functions
	- Area
	- Math PI - SELECT PI() 
	- ABS (Absolute valuie) - ABS(Value)
	- SQRT (Square root (the result will be float)) - SQRT(Value)
	- SQUARE (Raise to power of two) - SQUARE(Value)
	- FORMAT (Value, 'F2', 'bg-BG') - Round the number two digits after decimal point
		Example: FORMAT(PI(), 'F2', 'bg-BG'))
	- POWER (Raise the value to the desire Exponent - POWER(Value, Exponent)
	- ROUND - obtains the desire precision - ROUND(Value, Precision)
	- FLOOR & CEILING - return the nearest  integer - FLOOR(Value), CEILING(Value)
	- SIGN - returns 1, -1, 0, depending on the value of the sign - SIGN(Value)
	- RAND - gets random float value in range [0,1]. If Seed is not specified, it will be assigned randomly - RAND(), RAND(Seed)

4. Date Functions
	-DATEPART - extract a segemnt from a date as an integer. - DATEPART(Part, Date)
	-DATEDIFF - finds the difference between two dates. Part can be any part and format of date or time. - DATEDIFF(Part, FirstDate, SecondDate)
	-DATENAME - gets a string representation of date's part. - DATENAME(Part, Dateee)
	-DATEADD - performs date arithmetic - DATEADD(Part, Number, Date)
	-GETDATE - obtains the current date and time - SELECT GETDATE() - It returns both date and time. We can use FORMAT to takeee only date if we want.
	-EOMONTH - returns the last day of the month.
	
5. Other functions
	-CAST & CONVERT - conversion between data types. - CAST(Data AS NewType), CONVERT(NewType, Data)
	-ISNULL - swaps NULL values with a specified default value. - ISNULL(Data, DefaultValue)
	-COALESCE - evaluates the arguments in order and returns the current value of the first expression that initially does not evaluate to NULL.
		Example: SELECT COALESCE(NULL, NULL, 'third value', 'fourth value')
	-OFFSET & FETCH - get only specific rows from the result set. Used witth ODER BY for pagination.
		Example: SELECT ID, FirstName, LastName FROM Employees ORDER BY ID OFFSET 10 ROWS FETCH NEXT 5 ONLY	
		
6. Wildcards
	-They are used with WHERE for particular filtration. Similiar to regular expression but less capable.
	-Supported characters:
		-% - any string, including zero-length - 1 or more character
		-_ - any single characters
		- [...] - any character withun range
		- [^..] - any character not in the range
	-ESCAPE - specify a prefix to treat special characters as normal
		Example: SELECT ID, Name
				FROM Tracks
				WHERE Name LIKE '%max!%' ESCAPE '!'
